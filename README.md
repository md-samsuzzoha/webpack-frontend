# webpack-frontend
Guide to start with modern front-end tooling
<hr>
Install:</br>
  npm install</br>
  npm install webpack -g</br>
  npm install webpack-dev-server -g</br>
  webpack-dev-server
  
Run:
  localhost:8080 (depends on your running port)
  
Included:</br>
  webpack<br>
  sass<br>
  bootstrap
